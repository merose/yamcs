.. program:: yamcsadmin backup delete

**NAME**

    yamcsadmin backup delete - Delete a backup


**SYNOPSIS**

    ``yamcsadmin backup delete [--backupDir DIR] [--backupId ID]``


**OPTIONS**

    .. option:: --backupDir DIR

        Backup directory.

    .. option:: --backupId ID

        Backup ID.
